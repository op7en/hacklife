FinHelper

React-приложение для управления счетами из разных банков
Live Demo: https://op7en.github.io/hacklife/

Функции

Подключение счетов из VBank, ABank, SBank

Просмотр баланса и транзакций

Создание счетов (локально или через API)

Переводы между счетами

Премиум-функции с localStorage

Технологии

React, Hooks, Context API, Fetch API, Vite, CSS

Запуск локально
git clone https://github.com/op7en/hacklife.git
cd hacklife/react-app
npm install
npm run dev


Открой http://localhost:5173

На GitHub Pages демонстрация работает с мок-данными из-за CORS.
